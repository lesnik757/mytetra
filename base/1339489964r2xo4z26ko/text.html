Восстановить оч просто... Переходим с помощью браузера в указанную папку, т.е. в вашем случае http://ru.archive.ubuntu.com/ubuntu/dists/hardy-updates/ В ней находиться файл Release.gpg, сохраняем его на винт и далее возможны два варианта - использовать непосредственно gpg для испортирования ключа, или открыв synaptic выбрать пункт меню Настройка-Репозитории, появляется окно, в котором открываем вкладку Авторизация. Там жмем кнопку Импортировать файл ключа, выбираем наш файл и наслаждаемся



доступные ключи репозиториев и добавит их в систему. Основным достоинством этой команды является то, что таким образом можно найти и добавить несколько ключей за один раз.

Команда эта выглядит так:

sudo apt-key adv --recv-keys --keyserver keyserver.ubuntu.com `sudo aptitude update 2>&1 | grep -o '[0-9A-Z]\{16\}$' | xargs`
