Pidgin + X-Status


Иконки в tango стиле (tango круче)


1. Скачиваем последние (вот тут можно посмотреть последнюю версию)

wget http://heanet.dl.sourceforge.net/sourceforge/pidgin/pidgin-2.6.2.tar.bz2


2. Распаковываем скачанные исходники

tar xvfj pidgin-2.6.2.tar.bz2


3. Переходим в распакованную папку

cd pidgin-2.6.2


4. Скачиваем патч и иконки(в tango стиле) для x-status

wget http://developer.pidgin.im/raw-attachment/ticket/4508/xstatus-patch-pidgin-2.5.5.patch

wget http://developer.pidgin.im/raw-attachment/ticket/4508/custom_user_icons_tango.zip


5. Применяем скачанный патч:

patch -p1 < xstatus-patch-pidgin-2.5.5.patch


6. Распаковываем иконки

unzip custom_user_icons_tango.zip -d pidgin/pixmaps/emblems/16/


7. Устанавливаем необходимые для компиляции пакеты

sudo apt-get build-dep pidgin


8. Удаляем старую версию

sudo aptitude remove pidgin pidgin-data libpurple


9. Компилируем и устанавливаем pidgin

./configure --prefix=/usr

make

make install



Иконки в qip стиле


1. Скачиваем последние (вот тут можно посмотреть последнюю версию)

wget http://heanet.dl.sourceforge.net/sourceforge/pidgin/pidgin-2.6.2.tar.bz2


2. Распаковываем скачанные исходники

tar xvfj pidgin-2.6.2.tar.bz2


3. Переходим в распакованную папку

cd pidgin-2.6.2


4. Скачиваем патч и иконки(в tango стиле) для x-status

wget http://developer.pidgin.im/attachment/ticket/766/custom_icons4.patch?format=raw

wget http://developer.pidgin.im/attachment/ticket/766/custom_user_icons.tar.gz?format=raw


5. Применяем скачанный патч:

patch -p1 < custom_icons4.patch?format=raw


6. Распаковываем иконки

tar zxf custom_user_icons.tar.gz?format=raw -d pidgin/pixmaps/emblems/16/


7. Устанавливаем необходимые для компиляции пакеты

sudo apt-get build-dep pidgin


8. Удаляем старую версию

sudo aptitude remove pidgin pidgin-data libpurple


9. Компилируем и устанавливаем pidgin

./configure --prefix=/usr

make

make install


10. Копируем иконки

sudo cp custom_user_icons/* /usr/share/pixmaps/pidgin/emblems/16/

sudo chmod 644 /usr/share/pixmaps/pidgin/emblems/16/*
