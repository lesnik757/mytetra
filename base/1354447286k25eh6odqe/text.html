Как удалить старые ядра?

Ядра предыдущих версий не удаляются, чтобы на всякий случай оставалась возможность загрузиться в систему. Полностью удалить их можно после чистки средствами программы emerge, удалив папки с исходниками ядра в /usr/src/, модули в /lib/modules/ и само ядро в /boot/. Только сначала загрузитесь со свежим ядром. Удаление происходит примерно так:

emerge -C =sys-kernel/calculate-sources-2.6.38.4

rm -rf /usr/src/linux-2.6.38.4*

rm -rf /lib/modules/*2.6.38.4*

rm -rf /boot/*2.6.38.4*

rm -rf /boot/*old

Вместо первой команды можно использовать emerge -ac
