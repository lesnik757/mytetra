Файл подкачки устанавливается с помощью программы Swapper for Root. В программе указываем необходимое количество памяти, выбираем диск (обычно он один, SD Card EXT Partition), ставим галочку напротив Active Swap on Boot и нажимаемSave. Конечно же программе требуются права супер пользователя (root). После выполнения действий каждый раз при включение телефона файл будет активироваться как дополнительная оперативная память.

Раздел подкачки как и было написано ранее создается при помощи модифицированного загрузчика. По этому если ваш смартфон не от HTC, то есть вероятность потери гарантии. Для создания раздела подкачки нужен ClockworkMod Recovery.Заходим в рековери и переходим в advanced => partition sdcard. И первый раз вас спросят сколько место выделить под приложения, а второй раз сколько места выделить для раздела подкачки (максимум 256 мб). Дальше можно активировать раздел подкачки с помощью Программы Swapper for Root или установить bysybox и выполнить следующее:

1) Ставим Root Explorer (если еще нету)

2) Заходим в /system/etc/ и ищем файл init.qcom.post_boot.sh, ставим права (долгий тап по файлу, выбрать Permissions, поставить во 2 и 3 калонке галочки, потом OK)

3) Узнаем какой раздел отведен для SWAP:
В терминале прописываем: 
$ su
# busybox fdisk /dev/block/mmcblk0 

Пропускаем "The number of cylinders for this disk is set to 243328.
There is nothing wrong with that, but this is larger than 1024,
and could in certain setups cause problems with:
1) software that runs at boot time (e.g., old versions of LILO)
2) booting and partitioning software from other OSs
(e.g., DOS FDISK, OS/2 FDISK)"

В строчке Command (m for help): *
вместо * вводим p
должно получится так: Command (m for help): p 

Пропускаем "Disk /dev/block/mmcblk0: 7773 MB, 7973371904 bytes
4 heads, 16 sectors/track, 243328 cylinders
Units = cylinders of 64 * 512 = 32768 bytes" (инфа про флэшку: память и т.п)

Device Boot Start End Blocks Id System
/dev/block/mmcblk0p1 33 234848 7514112 b Win95 FAT32
Partition 1 does not end on cylinder boundary
/dev/block/mmcblk0p2 234849 243328 271360 82 Linux swap
Partition 2 does not end on cylinder boundary
Зеленым цветом выделен раздел который был у меня отведен для свап, у вас может отличаться ( пример: /dev/block/mmcblk0p3...4...5)

В строчке Command (m for help): * 
вместо * вводим q
должно получится так: Command (m for help): q

4) Заходим в терминал, прописываем:
echo "busybox swapon /dev/block/mmcblk0p2 (смотрим пунк 3)" >> /system/etc/init.qcom.post_boot.sh
писать полностью, со всеми значками! прописываем это сами!!!! 

5) Прописываем: reboot

6) Заходим в терминал, прописываем free
если СВАП не включился значит ошиблись при вводе, введите еще раз (будьте внимательны!)
