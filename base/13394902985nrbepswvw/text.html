Руководство по работе с wget.


Допустим нужно скачать с FTP определённую папку с подпапками, ну соответственно с файлами, находящимися в них.

Как реализовать это с помощью wget ?


wget -r ftp://www.aaa.com/files/*


Копирование сайта для локального просмотра:

wget -r -l0 -k http://www.vasyapupkin.com/


При этом будет включена рекурсивная выгрузка (ключ -r, --recursive),


-np --no-parent


Не подниматься выше начального адреса при рекурсивной загрузке
