1. Создайте пустой каталог в домашней директории: ~/.compose-cache

2. Создайте пустой каталог compose по пути: /var/cache/libx11/compose


Запустите на выполнение любую программу, после ее первого запуска в выше перечисленных каталогах создадутся файлы для ускоренного запуска этого приложения. По мере роботы с каждым другим приложением за некоторое время такие файлы будут для всех программ.


Это позволит запускать приложения Qt/KDE/Gtk на 50-150мс быстрее.


Справка:


В KDE при запуске любого приложения вызывается метод чтения информации из специального файла по пути: "/usr/share/X11/locale//Compose" . Этот файл имеет больше 5000 строк, чтение которого занимает некоторое процессорное время.

libX11 может создать такой же кэш, только для каждого приложения отдельно, что позволит считывать информацию быстрее.
