Для удаления лишних элементов со страницы можно воспользоваться специализированным сервисом PrintWhatYouLike. Перейдите на сайт при помощи браузера. В поле Enter URL введите адрес страницы, которую хотите распечатать. Нажмите Start.

4

В следующем окне откроется страница редактора. При помощи различных параметров панели слева вы можете отрегулировать настройки для печати – размер текста, стиль шрифта. При помощи пунктов Background, Images, Margins вы сможете убрать задний фон, картинки и ненужные параметры выравнивания. В правой части окна, при помощи прямоугольного выделения, выберите элементы, над которыми вы будете производить операции по изменению. При помощи кнопок Undo или Redo вы можете вернуть или повторить последние действия.

5

После окончания редактирования нажмите кнопку Print и дождитесь начала печати. Через пункт Save as вы можете сохранить измененный документ на компьютер.
