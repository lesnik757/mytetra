Отправка по ctrl + enter

Код для клавиш ctrl+enter:

Свернуть/Развернуть Исходный код


binding "my-bindings"

{

bind "Return" { "insert-at-cursor" ("\n") }

bind "Return" { "message_send" () }

}

widget "*pidgin_conv_entry" binding "my-bindings"


Код нужно сохранить в файл, который должен называться gtkrc-2.0 (именно так без расширения).


Затем файл нужно разместить в:


C:\Documents and Settings\trsteep\Application Data\.purple\


И на этом все


PS. Для nix систем это файл .gtkrc-2.0 в директории $HOME/
