Как установить Fallout 3 на Linux

fallou 3 linux

1) Качаем с winehq.org последнюю ,на данный момент, версию wine (1.1.8)

2) Забираем 2 патча для wine:

а) патч для исправления мыши bugs.winehq.org/attachment.cgi?id=17157

б) патч для исправления бага с текстурами (оторванные головы) bugs.winehq.org/attachment.cgi?id=17130


3) Распаковываем исходники wine и патчим программу (запускаем в каталоге с исходниками patch -p1 < «имя патча») , после чего собираем wine


4) В свежеустановленный wine (или в новую бутылку playonlinux) ставим с диска саму игру, после чего обязательно ломаем кряком (иначе проклятый старфорс не даст запустить в wine)


5)Устанавливаем DirectX 9 либо при помощи winetriks, либо старым добрым способом ( silverghost.org.ua/2008/03/23/directx-v-wine-skazhi-cedega-poka/ )


6)Производим изменения в реестре вайна:


а) в ветку HKEY_CURRENT_USER/Software/Wine/Direct3D добавляем:

“DirectDrawRenderer”=”gdi”

“OffscreenRenderingMode”=”fbo”

“PixelShaderMode”=”enabled”

“SoftwareEmulation”=”enabled”

“UseGLSL”=”enable”

“VertexShaderMode”=”hardware”

“VideoDescription”=”NVIDIA GeForce 9600 GT”

“VideoDriver”=”nv4_disp.dll”

“VideoMemorySize”=”512″


б) в ветку HKEY_CURRENT_USER/Software/Wine/DirectInput добавляем:

“MouseWarpOverride”=”enable”


7) Все! Спокойно запускаем игру и наслаждаемся игровым процессом :)
