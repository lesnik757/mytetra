Всему виной обновленный polkit.

Теперь старые конфиги необходимо переписывать., можно обойтись вот так:


# cat /etc/polkit-1/rules.d/10-udisks-allow-plugdev-all.rules

polkit.addRule(function(action, subject) {

if (action.id.indexOf("org.freedesktop.udisks.") == 0 &&

subject.isInGroup("plugdev")) {

return polkit.Result.YES;

}

});
